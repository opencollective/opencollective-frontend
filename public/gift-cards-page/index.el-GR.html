Δώστε το δώρο της συνεισφοράς.

Οι δωροκάρτες Open Collective δίνουν τη δυνατότητα στους υπαλλήλους ή τα μέλη της κοινότητάς σας να υποστηρίζουν τα έργα ανοιχτού κώδικα που αγαπούν.

Έχουμε φροντίσει για τα πάντα.

Απλώς πρέπει να μας δώσετε μια λίστα με διευθύνσεις ηλ. ταχυδρομείου και θα στείλουμε εμείς τους κωδικούς εξαργύρωσης. Μπορείτε επίσης να προσθέσετε και ένα προσωπικό μήνυμα. Θα σας χρεώσουμε με τον τρόπο πληρωμής σας μόνο όταν κάποια δωροκάρτα χρησιμοποιηθεί για δωρεά. Μπορείτε να περιορίσετε τις δωροκάρτες σε συγκεκριμένους τύπους συλλόγων (ανά ετικέτα ή οικοδεσπότη), ώστε να διατηρείτε τον έλεγχο των τύπων των συλλόγων που μπορούν να υποστηριχθούν με τα χρήματά σας. Κάθε μήνα θα λαμβάνετε ένα ενοποιημένο παραστατικό ανά οικοδεσπότη.

Κάντε το με τον δικό σας τρόπο.

Δημιουργήστε κωδικούς εξαργύρωσης και στείλτε τους με ηλ. ταχυδρομείο, δώστε τυπωμένες κάρτες ή διανείμετέ τες με όποιον τρόπο θέλετε.

Κάντε το κάτι παραπάνω.

Χρησιμοποιήστε το API μας για να δημιουργήσετε προβολές για την ιστοσελίδα σας ή άλλες δημιουργικές ιδέες. Δείτε για παράδειγμα τη σελίδα ανοιχτού κώδικα που έχει δημιουργήσει το Triplebyte.

Θα είστε σε καλή παρέα.

Δείτε πώς άλλοι οργανισμοί χρησιμοποιούν ήδη τις δωροκάρτες Open Collective.

Συχνές ερωτήσεις

Τι συμβαίνει αν οι δωροκάρτες δε χρησιμοποιηθούν;

Χρεώνεστε μόνο όταν χρησιμοποιηθούν οι δωροκάρτες. Αν δωρίσετε μια δωροκάρτα 50€ σε 10 από τους υπαλλήλους σας και μόνο 8 τη χρησιμοποιήσουν, θα χρεωθείτε μόνο 400€.

Θα λάβω παραστατικό;

Ναι, θα λάβετε ένα ορθά ενοποιημένο μηνιαίο παραστατικό από το Open Source Collective 501c6.

Υπάρχει ημερομηνία λήξης;

Από προεπιλογή οι δωροκάρτες λήγουν μετά από 3 μήνες αλλά μπορείτε να ορίσετε τη δικιά σας ημερομηνία λήξης. Εάν δε χρησιμοποιηθούν μέχρι την ημερομηνία λήξης, γίνονται ανενεργές και δε θα χρεωθείτε.

Έλαβα μία δωροκάρτα, πρέπει να τη χρησιμοποιήσω μονομιάς;

Όχι. Μπορείτε να χρησιμοποιήσετε τη δωροκάρτα σας για να δώσετε οποιοδήποτε ποσό σε οποιοδήποτε σύλλογο εφ' όσον έχετε αρκετό υπόλοιπο στην κάρτα.

Ποιος θα εμφανίζετε ως υποστηρικτής; Η εταιρεία μου ή ο υπάλληλος μου;

Και οι δύο! Το άτομο στο οποίο δώσατε τη δωροκάρτα είναι αυτό που διαλέγει τους συλλόγους τους οποίους θέλει να υποστηρίξει. Αλλά η δωρεά τους θα δείχνει ότι χρησιμοποίησαν μια δωροκάρτα δοσμένη από την εταιρεία σας και η εταιρεία σας θα εμφανίζετε επίσης ως χορηγός του συλλόγου. Βλέπε για παράδειγμα το Triplebyte που εμφανίζεται ως χορηγός του Electron.
