Σ' αυτές τις γιορτές,

δώστε στους προγραμματιστές σας το δώρο της συνεισφοράς.

Προσφέρετε στους προγραμματιστές σας 👩🏾‍💻👨🏼‍💻 μία δωροκάρτα για να κάνουν δωρεά στα πρότζεκτ ανοιχτού κώδικα που λατρεύουν. ❤️

Οι προγραμματιστές σας έκαναν απίστευτη δουλειά αυτόν τον χρόνο. Δε θα μπορούσαν να έχουν κάνει τόσα πολλά χωρίς την υποστήριξη της κοινότητας ανοιχτού κώδικα. Εκεί μαθαίνουν, εκεί μπορούν να επαναχρησιμοποιήσουν εργαλεία, βιβλιοθήκες, module ώστε να μη χρειάζεται να επανεφεύρουν τον τροχό και να εστιάσουν στο να τον κάνουν να δουλεύει για τις δικές σας ανάγκες.

Συχνές ερωτήσεις

Τι συμβαίνει αν οι δωροκάρτες δε χρησιμοποιηθούν;

Χρεώνεστε μόνο όταν χρησιμοποιηθούν οι δωροκάρτες. Αν δωρίσετε μια δωροκάρτα 50€ σε 10 από τους υπαλλήλους σας και μόνο 8 τη χρησιμοποιήσουν, θα χρεωθείτε μόνο 400€.

Θα λάβω παραστατικό;

Ναι, θα λάβετε ένα ορθά ενοποιημένο μηνιαίο παραστατικό από το Open Source Collective 501c6.

Υπάρχει ημερομηνία λήξης;

Από προεπιλογή οι δωροκάρτες λήγουν μετά από 3 μήνες αλλά μπορείτε να ορίσετε τη δικιά σας ημερομηνία λήξης. Αν δε χρησιμοποιηθούν μέχρι τότε, γίνονται ανενεργές. Δε θα χρεωθείτε αν δε χρησιμοποιηθούν ποτέ.

Έλαβα μία δωροκάρτα, πρέπει να τη χρησιμοποιήσω μονομιάς;

Όχι, μπορείτε να χρησιμοποιήσετε τη δωροκάρτα σας για να δώσετε οποιοδήποτε ποσό σε οποιοδήποτε σύλλογο εφ' όσον έχετε αρκετό υπόλοιπο στην κάρτα.

Ποιος θα εμφανίζετε ως υποστηρικτής; Η εταιρεία μου ή ο υπάλληλος μου;

Και οι δύο! Το άτομο στο οποίο δώσατε τη δωροκάρτα είναι αυτό που διαλέγει τους συλλόγους τους οποίους θέλει να υποστηρίξει. Αλλά η δωρεά τους θα δείχνει ότι χρησιμοποίησαν μια δωροκάρτα δοσμένη από την εταιρεία σας και η εταιρεία σας θα εμφανίζετε επίσης ως χορηγός του συλλόγου. Δείτε για παράδειγμα το Triplebyte εμφανίζετε ως χορηγός του Electron.

Δημιουργήστε έναν οργανισμό τώρα και αρχίστε να εκδίδετε δωροκάρτες!
